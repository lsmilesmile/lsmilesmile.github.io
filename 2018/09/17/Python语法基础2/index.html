2.2显示处理结果

如果要在屏幕输出内容，用函数print()，形式：

print(value, …, sep=‘ ’, end=‘\n’)

参数 的意义：

  • value是用户要输出的信息，后面的省略号表示可以有多个要输出的信息；
  • sep是多个要输出的信息之间的分隔符，默认为一个空格；
  • end是所有要输出信息之后添加的符号，默认是换行符。

注：如果要输出一个换行可以如下两种方式输出：
